# pagerViewTutorial-iOS
FSPageview 오픈 라이브러리를 활용한 이미지 슬라이드 튜토리얼

- 오픈라이브러리를 활용해서 이미지슬라이드뷰를 구성해보았다.
- pageController 의 index 를 이용해서 좌우측버튼을 통하 이미지전환과 그에따른 현재 index 의 image 를 보여줌으로써 동작시킴.
